# 연봉 실수령액 계산기 (Salary Calculator)

연봉을 입력하면 4대보험과 소득세를 자동 계산해 **월 실수령액**을 보여주는 웹 계산기입니다.
다중 입력값을 받아 항목별로 분해된 결과를 실시간으로 출력합니다.

## 데모

👉 **[https://leegentle.github.io/gentleSalaryCalculator/](https://leegentle.github.io/gentleSalaryCalculator/)**

> ⚠️ 4대보험 요율 및 근로소득세 간이세액표는 **2022년 기준**입니다.
> 본 프로젝트는 계산 로직 구현 예시로, 최신 요율 적용 시 `data.js`와 요율 상수만 갱신하면 됩니다.

## 주요 기능

- **연봉 입력 → 월 실수령액 자동 계산** (입력 즉시 실시간 반영)
- **공제 항목 분해 출력**: 국민연금 · 건강보험 · 장기요양보험 · 고용보험 · 근로소득세 · 지방소득세
- **다중 변수 반영**: 비과세액, 부양가족 수, 20세 이하 자녀 수
- **세전 월급 / 세후 월급 / 공제 합계** 동시 표시

## 계산 로직

각 공제 항목을 실제 요율 기준으로 산출합니다.

| 항목 | 계산 방식 |
|------|-----------|
| 국민연금 | 과세표준 × 4.5% (하한 35만 / 상한 553만 구간 적용) |
| 건강보험 | 과세표준 × 3.495% |
| 장기요양보험 | 건강보험료 × 12.27% |
| 고용보험 | 과세표준 × 0.9% |
| 근로소득세 | 간이세액표 조회 (과세표준 · 부양가족 · 자녀 수 기준) |
| 지방소득세 | 근로소득세 × 10% |

- **구간별 분기 처리**: 국민연금은 소득 하한·상한을 클램핑해 구간 외 값을 보정합니다.
- **외부 데이터 테이블 연동**: 근로소득세는 간이세액표 데이터(`data.js`)를 조회해 과세표준·부양가족 조건에 맞는 값을 반환합니다.
- **원 단위 절사 및 천 단위 콤마**: 모든 결과는 10원 단위로 절사하고 콤마를 적용해 표시합니다.

## 기술 스택

- **JavaScript (Vanilla)** — 프레임워크 없이 DOM API로 구현
- **모듈 시스템** — 세액 데이터를 별도 모듈(`data.js`)로 분리해 계산 로직과 데이터를 분리

## 구현 포인트

- 입력값 검증(숫자 외 입력 차단) 후 계산을 트리거하는 이벤트 흐름 구성
- 계산 로직과 데이터를 분리해, 세액표 갱신 시 `data.js`만 교체하면 되도록 설계
- 단일 `calculating()` 함수에서 모든 공제 항목을 일괄 산출해 결과를 갱신
